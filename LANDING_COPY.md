## Common waste patterns (section)

1. **Duplicate subscriptions** — ChatGPT Team next to Claude Team, or two API contracts covering the same workflows.  
2. **Oversized enterprise SKUs** — Business or Enterprise seats when headcount and compliance do not justify the uplift.  
3. **Unused or lumpy API spend** — Committed capacity and retail list pricing before credits, reserved instances, or brokered rates.  
4. **Overlapping coding tools** — Cursor + Copilot + legacy IDE add-ons — each defensible alone, expensive together.
